Connecting on Windows

  1. Download and install the official WireGuard client from wireguard.com/install.
  2. Click Download my VPN config above and save the .conf file somewhere you can find it.
  3. Open the WireGuard app, click Import tunnel(s) from file, and select the .conf file you downloaded.
  4. Select the imported tunnel in the list and click Activate.
  5. You're connected — challenge containers are now reachable at the addresses shown on each challenge's instance panel.

To disconnect, open WireGuard and click Deactivate on the tunnel.

Connecting on Linux

Install wireguard-tools for your distro, for example:

sudo apt install wireguard      # Debian / Ubuntu
sudo dnf install wireguard-tools    # Fedora
sudo pacman -S wireguard-tools      # Arch

Click Download my VPN config above, then bring the tunnel up with wg-quick:

sudo mv ~/Downloads/*.conf /etc/wireguard/wg0.conf
sudo wg-quick up wg0

Check it's up:

sudo wg show

To disconnect:

sudo wg-quick down wg0

Prefer a GUI? Most desktop environments can import the file directly — in GNOME/KDE network settings, add a new connection and choose Import from file (WireGuard), or run nmcli connection import type wireguard file your-config.conf if you use NetworkManager from the terminal.
